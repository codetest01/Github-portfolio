# App User Issues Analysis Framework

A structured product-management framework for analyzing mobile app user pain points using recent, credible, and evidence-backed feedback.

## Features
- PM-style problem statements
- User issue validation
- Source credibility filtering
- UX pattern identification
- Product prioritization
- Root cause analysis

## Use Cases
- Product management case studies
- UX audits
- Consumer app reviews
- Business school projects
- Product strategy research
